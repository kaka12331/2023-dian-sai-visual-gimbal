# 2023 全国大学生电子设计竞赛 — 视觉伺服双轴云台控制系统

基于 **STM32F103C8** 主控与 **嘉楠 K230** 智能视觉模组的实时目标跟踪云台方案。  
K230 负责目标检测与坐标输出，STM32 通过串口解析目标信息，使用 **PID 双环控制** 驱动云台舵机，使目标稳定保持在画面中心，并在 OLED 上实时显示目标、误差与舵机状态。

---

## 项目简介

本仓库为 2023 年全国大学生电子设计竞赛相关代码，包含：

- **STM32 控制端**（Keil µVision）：串口解析、PID、舵机、OLED  
- **K230 视觉端**（`Vision/2023电赛视觉.py`）：同心矩形 / 红绿激光检测、触摸调参、串口发坐标  

系统采用「视觉检测 + 串口通信 + 闭环控制」架构：

1. **K230** 完成图像采集与目标识别，按约定帧协议输出矩形/坐标数据；
2. **STM32** 通过 USART1 中断接收并解析数据；
3. 计算目标点与当前点的像素误差，分别对 **水平（Pan / Yaw）**、**俯仰（Tilt / Pitch）** 两轴做 PID 调节；
4. TIM2 输出 PWM 驱动云台舵机；
5. OLED 同步显示目标坐标、实际坐标、误差及舵机角度，便于现场调试。

## 核心功能说明

### 0. K230 视觉端（`Vision/2023电赛视觉.py`）

运行平台：**庐山派 K230**（CanMV / media.sensor + ST7701 触屏 + UART2）。

| 模式 (flag) | 功能 | 说明 |
|-------------|------|------|
| 0 | 主菜单 | 开始 / 常用调节 / 红绿 LAB / 功能1↔功能2 切换 |
| 1 | 功能1：同心矩形 + 红激光 | 检测内外矩形中线路径，红激光为实际点，串口发坐标 |
| 2 | 常用阈值调节 | 调 `L_min/L_max`、`roi_s`，二值化实时预览 |
| 3 | 红绿 LAB 调节 | 分别调节红/绿激光 LAB 阈值并预览 |
| 4 | 功能2：红 + 绿激光 | 同时检测红绿点并串口发送 |

**硬件（K230）：**

| 接口 | 配置 |
|------|------|
| 摄像头 | Sensor id=2，640×360 RGB565 |
| 显示 | ST7701，800×480（左侧画面 + 右侧按钮） |
| 串口 | UART2，TX=GPIO11，RX=GPIO12，**115200** |
| 触屏 | TOUCH(0) |

**串口发送内容（K230 → STM32）：**

| 模式 | x1, y1 | x2, y2 |
|------|--------|--------|
| 功能1 | 矩形路径当前点 | 红激光坐标 |
| 功能2 | 红激光 | 绿激光（缺失时复用另一点） |

路径插值：四边形每条边插入 `PATH_N=30` 点，按 `PATH_INTERVAL=100ms` 轮询发送，便于云台沿矩形轮廓跟踪。

部署：将脚本拷到 K230 可运行路径（如 `main.py`），接好 UART2 与 STM32 USART1 后上电运行。

### 1. K230 ↔ STM32 串口通信协议

一帧格式（共 12 字节）：

| 字段 | 内容 | 说明 |
|------|------|------|
| 帧头 | `0xAA` | 起始标志 |
| 数据 | 10 字节 | `x1,y1,x2,y2,z` 各 2 字节，大端有符号 16 位 |
| 帧尾 | `0x55` | 结束标志 |

- **z**：附加量（预留，默认 0）
- K230：`struct.pack(">BhhhhhB", 0xAA, x1, y1, x2, y2, z, 0x55)`
- STM32：`K230_UART_Handler` 状态机解析；`main.c` 中 `actual` 取自 `x1/y1`，`target` 取自 `x2/y2`

### 2. 双轴 PID 闭环跟踪

主循环检测到新帧后：

```text
err = target - actual
pan_adj  = PID_pan (target_x, actual_x)   → 水平舵机角度增量
tilt_adj = PID_tilt(target_y, actual_y)   → 俯仰舵机角度增量
```

默认参数（可按现场再调）：

| 轴 | Kp | Ki | Kd | 积分限幅 | 输出限幅 |
|----|----|----|-----|----------|----------|
| Pan（水平） | 0.012 | 0 | 0.005 | 10 | 2.0°/次 |
| Tilt（俯仰） | 0.005 | 0 | 0.005 | 10 | 2.0°/次 |

特点：

- 积分抗饱和（`max_integral`）
- 单次输出限幅（`max_output`），避免舵机猛转丢目标
- 角度软件限幅：Pan 0°~360°，Tilt 0°~180°

### 3. 舵机 PWM 驱动

- 定时器：**TIM2**，PSC=71，ARR=19999 → **1 MHz 计数，20 ms 周期（50 Hz）**
- **PA0 / TIM2_CH1**：水平舵机（Pan）
- **PA1 / TIM2_CH2**：俯仰舵机（Tilt）
- 脉宽映射：约 **0.5 ms ~ 2.5 ms** 对应机械行程两端

### 4. OLED 实时调试界面

| 行 | 含义 |
|----|------|
| T: X=… Y=… | Target 目标坐标 |
| A: X=… Y=… | Actual 实际坐标 |
| E: X=… Y=… | Error 像素误差（带符号） |
| S: P=… T=… | Servo 当前 Pan / Tilt 角度 |

静态标签只初始化一次，数值局部刷新，减轻闪屏。

## 硬件连接参考

| 功能 | STM32 引脚 | 备注 |
|------|------------|------|
| 舵机 Pan | PA0（TIM2_CH1） | 水平轴 |
| 舵机 Tilt | PA1（TIM2_CH2） | 俯仰轴 |
| 串口 TX | PA9（USART1_TX） | 可选回传 |
| 串口 RX | PA10（USART1_RX） | 接 K230 TX |
| OLED | 依 `OLED.c` 接线 | 常用 I2C/SPI 软件模拟 |
| 供电 | 舵机建议独立供电 | 共地，注意电流 |

**注意：**

1. K230 与 STM32 共地，电平匹配（3.3 V 逻辑）。
2. 舵机电流较大，勿仅靠 MCU 板 USB 供电。
3. 若跟踪方向相反，在 `main.c` 中将 `+=` 改为 `-=`（或调整 PID 符号）。

## 编译与下载

### 环境要求

- Keil µVision 5（工程标注 ARM Compiler V5.06）
- 器件包：`Keil.STM32F1xx_DFP`（工程使用 2.4.0）
- 下载器：ST-Link / J-Link 等（按实际配置）

### 步骤

1. 用 Keil 打开 `Project.uvprojx`
2. 确认 Target 器件为 **STM32F103C8**
3. 编译（Rebuild）
4. 连接调试器，下载至 Flash
5. 接通 K230 串口与云台电源，观察 OLED 与跟踪效果

### 调参建议

1. 先只开 P，使云台缓慢对准目标；
2. 再加少量 D 抑制振荡；
3. 积分慎用，本工程默认 Ki=0；
4. 输出限幅过小跟踪偏慢，过大易过冲丢目标。
